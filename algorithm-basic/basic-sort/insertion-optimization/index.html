# 插入排序写法二：先暂存再后移

「将一个数字插入一个有序的数组」这一步，可以不使用逐步交换，使用先赋值给「临时变量」，然后「适当的元素」后移，空出一个位置，最后把「临时变量」赋值给这个空位。

其实这种插入方式更像插入排序本来的样子。《算法导论》上的图更形象。
